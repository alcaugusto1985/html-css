Principais formatações:

Negrito:

O negrito pode ser usado com dois tipos de tags diferentes, sendo uma dando um sentido semântico a ela.

Podemos utilizar a tag < b >, mas esta não dá nenhum sentido ao uso.

Podemos também utilizar a tag < strong >, porém, esta sim da um significado a sua utilização.

Itálico

Assim como ocorre com as tags < b > e < strong >, o itálico funciona da mesma forma.

Podemos utilizar a tag < i >, mas assim como ocorre com a tag < b >, ela também não da um significado semantico ao html.

Mas também podemos utilizar a tag < em >, essa sim tem um significado para o html.

Outras formatações:

Um texto pré formatado é representado pela tag < code >, e é muito utilizado para representar trechos de código fonte, como mostra o exemplo abaixo.

Ex: document.querySelectorAll('.teste'), é um comando em javascript.

Também podemos representar um cód identado em várias linhas utilizando a tag < pre >.

Citação simples:

Citações simples são representadas pela tag < q >, e é utilizada para recitar ditados populares, por exemplo.

Ex: Como diz a minha avó: Saco vazio não para em pé.

Desta forma, a citação fica representada dentro de aspas.

Citações completas são usadas para referênciar algum trecho de livro, por exemplo.

É representada pela tag < blockquote >.
