Trabalhando com listas:

Listas ordenadas:

São representadas em ordem numérica, alfabética, ou algarismos romanos

Ex:

  1. Acordar
  2. Tomar banho
  3. Tomar café
  4. Estudar
  5. Ir trabalhar
  6. Voltar do trabalho
  7. Jantar
  8. Assistir tv

Listas não ordenadas:

São representadas por pontos e não possuem uma ordem necessária.

Pode ser usado com uma listas de compras, por exemplo.

Ex:

  • Açucar
  • Sal
  • Limão
  • Arroz
  • Feijão
  • Leite

As listas também podem ser utilizadas misturadas (ordenadas e não ordenadas).

Ex:

Listas de definção:

São listas criadas para dar definição a alguma palavra específica:

As tags utilizadas são: < dl > (tag principal), < dt > (tag com a palavra a ser definida) e < dd > (Tag com a definição da palavra contida na tag < dt >).

HTML5:
Linguagem de marcação de texto conhecida como Hypertext Markup Language
CSS3:
Linguagem de estilização de sites conhecida como Cascading style sheets
Javascript:
Linguagem de programação utilizada para criar interatividade nas páginas
